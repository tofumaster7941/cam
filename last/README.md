# 3D Suit Mimic App

This is a real-time face-tracking application that overlays a 3D suit (Iron Man or Robot) on the user's face.

## How to use

1.  **Open the folder** in your IDE (e.g., VS Code or Cursor).
2.  **Start the local server**:
    ```bash
    python run_server.py
    ```
3.  **Open the App**:
    Go to [http://localhost:8000](http://localhost:8000) in your browser.

## Controls

- **Select Mode**: Use the buttons on the landing page to choose between "Normal" (Iron Man) and "Black Mode" (Dense Nodes).
- **Spacebar**: In "Black Mode", use the spacebar to toggle between the camera view and a dark background with just the glowing nodes.
- **Hands Up!**: Perform a hands-up gesture during the game to trigger a special event.

## Requirements

- A modern web browser.
- Python 3 installed (to run the server).
- MacBook camera (or any built-in camera).
